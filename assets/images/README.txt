====================================
DEVELOPER PHOTO INSTRUCTIONS
====================================

To add your photo to the Developer Info page:

1. Add your photo file to this folder (assets/images/)
2. Name it: developer_photo.png
   (or developer_photo.jpg)
3. Recommended size: 400x400 pixels or square aspect ratio
4. The app will automatically load your photo

Your photo will be displayed on the Developer Info page.

Note: If you don't add a photo, a default icon will be shown.
